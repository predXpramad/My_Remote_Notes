# 1. Git Setup Commands

| Command            | Syntax                                               | Usage                                     |
| ------------------ | ---------------------------------------------------- | ----------------------------------------- |
| Check Git Version  | `git --version`                                      | Displays the installed Git version.       |
| Configure Username | `git config --global user.name "Your Name"`          | Sets the global Git username.             |
| Configure Email    | `git config --global user.email "email@example.com"` | Sets the global Git email.                |
| View Config        | `git config --list`                                  | Shows all Git configuration settings.     |
| Edit Config        | `git config --global --edit`                         | Opens Git configuration file for editing. |

---

# 2. Repository Creation Commands

| Command                  | Syntax                           | Usage                                              |
| ------------------------ | -------------------------------- | -------------------------------------------------- |
| Initialize Repository    | `git init`                       | Creates a new Git repository in current directory. |
| Clone Repository         | `git clone repository_url`       | Downloads an existing repository.                  |
| Clone to Specific Folder | `git clone repo_url folder_name` | Clones repo into specified directory.              |

---

# 3. Basic Workflow Commands

| Command             | Syntax                    | Usage                                  |
| ------------------- | ------------------------- | -------------------------------------- |
| Check Status        | `git status`              | Shows file changes and staging status. |
| Add File            | `git add file_name`       | Adds file to staging area.             |
| Add All Files       | `git add .`               | Adds all modified files.               |
| Commit Changes      | `git commit -m "message"` | Saves staged changes to repository.    |
| View Commit History | `git log`                 | Displays commit history.               |
| Short Log           | `git log --oneline`       | Shows compact commit history.          |
| Show Changes        | `git diff`                | Displays changes in files not staged.  |
| Show Staged Changes | `git diff --staged`       | Displays staged changes.               |

---

# 4. Branch Management Commands

| Command                | Syntax                        | Usage                               |
| ---------------------- | ----------------------------- | ----------------------------------- |
| List Branches          | `git branch`                  | Displays all local branches.        |
| Create Branch          | `git branch branch_name`      | Creates a new branch.               |
| Switch Branch          | `git checkout branch_name`    | Switches to another branch.         |
| Create & Switch Branch | `git checkout -b branch_name` | Creates and switches to new branch. |
| Rename Branch          | `git branch -m new_name`      | Renames current branch.             |
| Delete Branch          | `git branch -d branch_name`   | Deletes a branch.                   |
| Force Delete Branch    | `git branch -D branch_name`   | Deletes branch forcefully.          |

---

# 5. Remote Repository Commands

| Command         | Syntax                           | Usage                                 |
| --------------- | -------------------------------- | ------------------------------------- |
| View Remotes    | `git remote -v`                  | Lists remote repositories.            |
| Add Remote      | `git remote add origin repo_url` | Adds remote repository.               |
| Remove Remote   | `git remote remove origin`       | Removes remote repository.            |
| Fetch Changes   | `git fetch`                      | Downloads changes from remote.        |
| Pull Changes    | `git pull origin branch_name`    | Fetches and merges remote changes.    |
| Push Changes    | `git push origin branch_name`    | Uploads commits to remote repository. |
| Push First Time | `git push -u origin branch_name` | Sets upstream branch and pushes.      |

---

# 6. Undo & Reset Commands

| Command           | Syntax                     | Usage                                           |
| ----------------- | -------------------------- | ----------------------------------------------- |
| Undo File Changes | `git checkout file_name`   | Reverts file to last commit.                    |
| Unstage File      | `git reset file_name`      | Removes file from staging area.                 |
| Reset Soft        | `git reset --soft HEAD~1`  | Undo commit but keep changes staged.            |
| Reset Mixed       | `git reset --mixed HEAD~1` | Undo commit and unstage changes.                |
| Reset Hard        | `git reset --hard HEAD~1`  | Completely removes commit and changes.          |
| Revert Commit     | `git revert commit_id`     | Creates new commit that undoes previous commit. |

---

# 7. Stashing Commands

| Command       | Syntax                    | Usage                                  |
| ------------- | ------------------------- | -------------------------------------- |
| Save Changes  | `git stash`               | Temporarily saves uncommitted changes. |
| List Stashes  | `git stash list`          | Shows saved stashes.                   |
| Apply Stash   | `git stash apply`         | Applies latest stash.                  |
| Pop Stash     | `git stash pop`           | Applies and removes stash.             |
| Drop Stash    | `git stash drop stash_id` | Deletes a stash.                       |
| Clear Stashes | `git stash clear`         | Removes all stashes.                   |

---

# 8. Merging & Rebasing Commands

| Command         | Syntax                   | Usage                                        |
| --------------- | ------------------------ | -------------------------------------------- |
| Merge Branch    | `git merge branch_name`  | Combines another branch into current branch. |
| Rebase Branch   | `git rebase branch_name` | Reapplies commits on top of another branch.  |
| Abort Merge     | `git merge --abort`      | Cancels merge process.                       |
| Abort Rebase    | `git rebase --abort`     | Cancels rebase process.                      |
| Continue Rebase | `git rebase --continue`  | Continues rebase after conflict resolution.  |

# 9. Tagging Commands

# 11. Advanced Git Commands

| Command               | Syntax                                     | Usage                                      |
| --------------------- | ------------------------------------------ | ------------------------------------------ |
| Cherry Pick Commit    | `git cherry-pick commit_id`                | Applies specific commit to current branch. |
| Clean Untracked Files | `git clean -f`                             | Removes untracked files.                   |
| Show Reference Logs   | `git reflog`                               | Shows history of HEAD changes.             |
| Archive Repo          | `git archive --format=zip HEAD > repo.zip` | Creates archive of repository.             |
| Submodule Add         | `git submodule add repo_url`               | Adds external repository as submodule.     |
| Submodule Update      | `git submodule update --init`              | Initializes submodules.                    |

# 12. Git Workflow Summary

Typical workflow:

1️⃣ Initialize repository

```
git init
```

2️⃣ Add files

```
git add .
```

3️⃣ Commit changes

```
git commit -m "Initial commit"
```

4️⃣ Connect to remote

```
git remote add origin repo_url
```

5️⃣ Push code

```
git push -u origin main
```
